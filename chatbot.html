WhatsApp Chatbot for JSMobile

A custom WhatsApp automation built with Node.js (whatsapp-web.js), integrated with Google Sheets for logging and categorization. Designed to reduce manual handling of user inquiries and improve response consistency.

Role: Developer
Platform: WhatsApp / Google Sheets
Year: 2025
Node.js whatsapp-web.js Google Sheets Apps Script
View Repo JSMobile App

Project Overview

The chatbot handles incoming messages with an auto-reply menu (options 1 / 2 / 3), captures and logs messages to Google Sheets for later review, and supports storing user names by phone number. The architecture is modular so future integration with AI (OpenAI API) can be added with minimal changes.

Key Features

  • Auto-reply menu system (menu options & guided flows)
  • Message logging to Google Sheets (category & timestamp)
  • User data handling (save name per phone number)
  • Planned AI integration (OpenAI) for intelligent replies
  • Error handling & reconnection logic for WhatsApp Web sessions

Tech & Tools

  • Node.js, whatsapp-web.js
  • Google Sheets + Apps Script (Webhook)
  • ngrok / webhook tunnel (optional for local testing)
  • GitHub for source control

// Example: simple auto-reply handler (Node.js + whatsapp-web.js)
client.on('message', async message => {
  const from = message.from;
  const text = message.body.trim();
  if (text === '1') {
    client.sendMessage(from, 'Thanks! Please type your issue...');
  } else if (text === '2') {
    client.sendMessage(from, 'For info, visit: https://jsmobile.org');
  } else {
    client.sendMessage(from, 'Welcome! Reply 1 for Support, 2 for Info, 3 for Sales.');
  }
});

Impact / Outcome

Reduced manual workload for handling FAQs and improved tracking of user requests. The Google Sheets log enables supervisors to categorize and prioritize follow-ups.
